# 📘 Student Management API (Project 2)

## 🚀 Project Overview
This is a simple Student Management REST API built using Node.js and Express.js.  
It demonstrates backend CRUD operations (Create, Read, Update, Delete).

---

## ⚙️ Tech Stack
- Node.js  
- Express.js  
- JavaScript  

---

## 📌 API Endpoints

GET /students  
GET /students/:id  
POST /students  
PUT /students/:id  
DELETE /students/:id  

---

## 🧪 Input Validation
- Name is required  
- Email is required  
- Age must be positive  

---

## 📡 Status Codes
- 200 OK  
- 201 Created  
- 400 Bad Request  
- 404 Not Found  

---
## Live API Link

https://student-managementapi-c63r.onrender.com/students


## ▶️ How to Run

npm install  
npm start  

Server runs at:
http://localhost:3000
