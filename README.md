# DecodeTech Solutions - Responsive Frontend Website

## Project Overview
This project is a responsive frontend website developed as part of the Full Stack Development Internship. The website represents a fictional technology company named **DecodeTech Solutions** and demonstrates modern web design principles using HTML, CSS, and JavaScript.

## Features
- Responsive design for mobile, tablet, and desktop devices
- Modern and clean user interface
- Hero section with company introduction
- About Us section
- Services section
- Team section
- Statistics/Counter section
- Contact section
- Social media links
- Smooth navigation

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Project Structure

project-folder/
│
├── index.html
├── style.css
├── script.js
└── README.md

## How to Run
1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in any web browser.

## Learning Outcomes
- Building responsive web pages
- Using HTML for structure
- Styling with CSS
- Adding interactivity with JavaScript
- Creating user-friendly layouts
