# Decode Labs Internship - Project 4
## Frontend & Backend Integration

### 📖 Project Overview
This project demonstrates the integration of a frontend application with a backend REST API. It allows users to view and add student records through a web interface. The backend is built using Node.js, Express.js, and MongoDB Atlas, while the frontend is developed using HTML, CSS, and JavaScript.

---

## ✨ Features

- View all students
- Add a new student
- REST API integration using Fetch API
- MongoDB Atlas database
- Responsive user interface
- Full Stack Integration

---

## 🛠️ Technologies Used

### Frontend
- HTML5
- CSS3
- JavaScript (Fetch API)

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- CORS

## 🚀 API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | /students | Get all students |
| POST | /students | Add a new student |
| PUT | /students/:id | Update a student |
| DELETE | /students/:id | Delete a student |

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/ATHULKRISHNAN48/DecodeLabs-Internship.git
```

### Navigate to the project

```bash
cd decodelabs_task4
```

### Install dependencies

```bash
npm install
```

### Run the server

```bash
node server.js
```

### Open in your browser

```
http://localhost:3000
```
